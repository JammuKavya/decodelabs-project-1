🤖 Smart Chatbot

📖 Project Overview

Smart Chatbot is a beginner-friendly, rule-based chatbot developed in Python. It interacts with users through a command-line interface and responds to predefined commands. The project demonstrates core Python programming concepts such as loops, conditional statements, lists, dictionaries, string manipulation, and the use of built-in modules like "datetime" and "random".

This project was built to strengthen Python fundamentals and gain hands-on experience in creating interactive applications.

---

✨ Features

- 👋 Greets users with a welcome message
- ⏰ Displays the current time
- 📅 Shows the current date
- 💡 Generates random motivational quotes
- 🔄 Reverses user-entered text
- 📜 Maintains and displays chat history
- ℹ️ Provides chatbot information through an "About" section
- ❌ Handles unknown commands with a friendly response
- 🚪 Allows users to exit the chatbot safely

💬 Available Commands

Command| Description
"hello"| Greets the user
"time"| Displays the current time
"date"| Displays today's date
"quote"| Shows a random motivational quote
"reverse"| Reverses the entered text
"history"| Displays previous user inputs
"about"| Displays chatbot information
"help"| Shows the list of available commands
"exit"| Closes the chatbot
